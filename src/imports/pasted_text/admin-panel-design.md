Design multiple admin panel pages for a car wash management system. Use a consistent modern SaaS admin dashboard style across all pages.

Global Layout for all pages:

* Left sidebar navigation
* Top header with page title, search bar, notifications icon, and admin profile
* Light background
* White content cards
* Rounded corners
* Soft shadows
* Blue primary color
* Clean modern dashboard UI

Sidebar menu items:

* Dashboard
* Bookings
* Calendar / Schedule
* Customers
* Vehicles
* Services
* Pricing
* Promotions
* Loyalty Program
* Staff / Drivers
* Branches
* Payments
* Reports
* Settings

Design the following pages:

---

1. Customers Page

---

Page title: Customers

Include:

* Search customer
* Filter by loyalty status
* Filter by total bookings
* Customers table with columns:

  * Customer ID
  * Name
  * Phone
  * Email
  * Vehicles
  * Total Bookings
  * Loyalty Points
  * Last Booking Date
  * Actions (View Profile)

Customer profile view should include:

* Customer info
* Vehicles list
* Booking history
* Payments history
* Loyalty rewards

---

2. Services Page

---

Page title: Services

Include:

* Button: Add Service
* Services table with:

  * Service Name
  * Description
  * Duration
  * Price (Sedan)
  * Price (SUV)
  * Price (Truck)
  * Status (Active/Inactive)
  * Actions (Edit/Delete)

Service form should include:

* Service name
* Description
* Duration
* Pricing per vehicle type
* Add-ons
* Active toggle
